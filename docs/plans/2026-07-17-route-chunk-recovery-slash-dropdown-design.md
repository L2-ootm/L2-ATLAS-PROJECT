# Route Chunk Recovery and Upward Slash Dropdown

## Failure contract

An open ATLAS page may retain a lazy-route hash after a new WebUI build removes that file. The SPA fallback then answers the missing `/assets/*.js` request with `index.html` (`200 text/html`), which the browser refuses as a module. Every lazy route therefore gets one guarded document reload. A session marker prevents loops; a second failure renders an ATLAS-owned recovery surface instead of React Router's developer error page.

## Slash interaction

The inline command index belongs above the composer, matching the TUI's completion geometry. It is a sibling of the clipped composer shell, anchored to the shell's upper edge and revealed upward. Arrow navigation, Tab completion, Enter behavior, command sources, queue semantics, and the local-draft performance boundary remain unchanged. The dropdown uses the existing cockpit surface ladder, one active rail, a compact instruction header, and reduced-motion fallback.
