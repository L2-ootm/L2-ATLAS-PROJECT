# ATLAS v0.1.0 — Open Research Preview

ATLAS is an auditable AI operator cockpit that brings missions, agents, tools,
knowledge, approvals, and operational evidence into one local runtime.

![ATLAS npm installation](https://raw.githubusercontent.com/L2-ootm/L2-ATLAS-PROJECT/main/docs/media/atlas-npm-install.png)

## Install

Windows x64 requires Node.js 20 or newer:

```powershell
npm install --global @systemsl2/atlas
atlas db init
atlas doctor
atlas up --services gateway,cockpit
```

The npm launcher and runtime are also available directly from the release assets
for audit and provenance verification.

## What ships

- dependency-free npm lifecycle launcher with update, rollback, versions, doctor,
  and uninstall commands;
- self-contained Windows x64 runtime with embedded Python 3.13.11;
- Rust gateway and Go terminal UI;
- compiled React WebUI cockpit with chat, mission control, runs, ledger, models,
  integrations, system health, and slash-command palette;
- Hermes-derived agent foundation, ATLAS runtime services, wiki, migrations, and
  bundled modules;
- immutable application versions separated from persistent `ATLAS_HOME` state and
  user-created modules.

## Verified release gates

- anonymous public-registry installation and exact platform dependency resolution;
- 23 database migrations and default model bootstrap;
- runtime checksum doctor and no-op update verification;
- user-module preservation and zero development-checkout mutation;
- packaged gateway execution and cockpit HTTP 200 smoke test;
- full publishable-history secret scan with no findings.

## Current boundaries

- the self-contained npm runtime currently supports Windows x64;
- macOS and Linux use the source setup path for this preview;
- a real cross-version npm update/rollback will be exercised with the next release;
- the embedded runtime may display `0.1.0+dev` because source-mode package metadata is
  not installed; the distributed npm package version is `0.1.0`.

ATLAS is an open research preview. Do not use it with sensitive production data
until its security and operational boundaries fit your environment.
