<!-- API response shapes from native/atlas-core-rs/crates/atlas-gateway/src/db.rs and lib.rs -->
<!-- All endpoints are on 127.0.0.1:8484 -->

Gateway endpoints (Phase 7, live):
  GET  /health                            → { status, service, version, db }
  GET  /v1/missions?limit=N               → { missions: Mission[], count: number }
  GET  /v1/missions/{id}                  → { mission: Mission, runs: Run[] }
  POST /v1/missions                       → 201 { mission: Mission, runs: [] }
  POST /v1/missions/{id}/run              → 201 { run: Run }
  GET  /v1/runs/{id}                      → { run: Run }
  GET  /v1/runs/{id}/events?after=N       → { run_id, events: AuditEvent[], next_cursor: N }
  GET  /v1/runs/{id}/stream               → SSE stream, event:"audit" data:JSON, event:"end", event:"error"
  GET  /v1/wiki/pages?limit=N             → { pages: WikiPage[], count: number }
  GET  /v1/wiki/search?q=STR              → { query: string, results: WikiPage[] }

TypeScript types (derive from db.rs mission_row / run_row):
  Mission { id: string; title: string; intent: string; status: string; project: string; created_at: string; updated_at: string }
  Run { id: string; mission_id: string; session_id: string|null; status: string; started_at: string; finished_at: string|null; summary: string }
  AuditEvent { rowid: number; run_id: string; event_type: string; payload: string; created_at: string }
  WikiPage { slug: string; title: string; body: string; layer: number; created_at: string; updated_at: string }
</interfaces>
