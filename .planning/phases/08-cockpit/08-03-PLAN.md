| URL params → getMission() | Mission ID from URL path passed to API call |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-09 | Tampering | CreateMissionModal title/intent inputs | accept | Data passes through gateway → atlas CLI which has policy engine; frontend has no direct DB access; input goes to atlas --title/--intent args (no shell injection) |
| T-08-10 | Spoofing | mission ID in URL path | accept | Loopback-only, single operator v1.0; gateway returns 404 for unknown IDs |
| T-08-11 | Information Disclosure | optimistic UI showing pending mission before API confirms | accept | Pending row is immediately replaced by API response; no data persisted client-side beyond session |
