<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| EventSource → gateway SSE | SSE stream from loopback; events contain user-generated content (payloads) displayed in DOM |
| Payload display | Audit event payload rendered as text in DOM — no innerHTML or dangerouslySetInnerHTML |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-12 | Tampering | SSE payload display | mitigate | Payload rendered as Svelte text content (not innerHTML); Svelte auto-escapes; no XSS vector |
| T-08-13 | Denial of Service | DOM unbounded growth from SSE events | mitigate | Hard cap at 500 rows — splice(0, events.length - 500) enforced on every insert |
| T-08-14 | Denial of Service | JSONL export large payload | accept | Export is operator-initiated; single operator local; no server-side exposure |
| T-08-15 | Spoofing | EventSource reconnect loop | mitigate | Single retry only; if retry fails, static error message shown — no infinite reconnect loop |
| T-08-SC | Tampering | No new npm packages in this plan | accept | No new packages installed |
</threat_model>
