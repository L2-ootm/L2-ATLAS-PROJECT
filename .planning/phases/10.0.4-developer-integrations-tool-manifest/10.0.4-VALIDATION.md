## Sampling Rate

- **After every task commit:** Run the quick run command (the touched module's test file).
- **After every plan wave:** Run the full Python suite + `cargo test -p atlas-gateway` if Rust changed + `npm run build` if cockpit changed.
- **Before `/gsd-verify-work`:** Full suite green (Python + Rust + cockpit tsc/build).
- **Max feedback latency:** 30 seconds.
