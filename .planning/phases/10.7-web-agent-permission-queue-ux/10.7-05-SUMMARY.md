# Plan 10.7-05 Summary

The React Console is now a first-class client of the shared surface-session and normalized-event
contracts. A standard-hooks coordinator survives route changes, restores reconnect identity,
resumes suspended sessions, heartbeats ownership, replays ordered events, and fetches only its
owned approval queue.

Console submissions create missions and surface-bound runs through the coordinator. All ten
normalized event kinds project visibly into the existing workbench. Malformed/unknown events
cross an explicit error boundary. Production React no longer exports or invokes
`consoleChatStream`, and System approval history is read-only.
