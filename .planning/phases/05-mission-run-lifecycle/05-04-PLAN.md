| CLI -> SQLite (read-only queries only) | status and cancel subcommands read directly from DB for UX; no mutations in CLI layer |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-08 | Tampering | CLI status SELECT query | accept | Read-only SELECT with parameterized mission_id — same rationale |
