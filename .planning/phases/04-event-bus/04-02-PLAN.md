    Private helper _redact(text: str) -> str:
      Apply each pattern in SECRET_PATTERNS using pat.sub(lambda m: m.group(0).replace(m.group(2), "[REDACTED]"), text). Return the fully-redacted string. Called on every data, args, result string before model construction.

    Public function emit(conn, lock, *, run_id, event_type, task_id=None, session_id=None,
                         tool_call_id=None, tool_name=None, data=None, duration_ms=None,
                         policy_result=None, tool_call_kwargs=None) -> AuditEvent:
      1. Serialize data: data_str = _redact(json.dumps(data or {}))
      2. Construct AuditEvent(**fields) — this is where Pydantic validates event_type; ValidationError propagates to caller (HB-04-02).
      3. row = event.model_dump() — JSON-safe dict (datetime already serialized by field_serializer).
      4. If tool_call_kwargs is not None: build tc_kwargs dict; for each of "args" and "result" in tc_kwargs, if value is dict call json.dumps() then _redact(); if value is str call _redact(); if None use "{}" for args and None for result. Construct ToolCall(audit_event_id=event.id, run_id=run_id, **tc_kwargs).
      5. Acquire lock then use sqlite3 context manager for atomic write:
           with lock:
               with conn:
                   conn.execute("INSERT INTO audit_events VALUES (:id,:run_id,:task_id,:session_id,:tool_call_id,:event_type,:tool_name,:timestamp,:duration_ms,:data,:policy_result)", row)
                   if tc is not None:
                       conn.execute("INSERT INTO tool_calls VALUES (:id,:audit_event_id,:run_id,:tool_name,:args,:result,:exit_code,:stdout,:stderr,:duration_ms,:policy_allowed,:requires_approval,:timestamp)", tc.model_dump())
      6. Return event.
      NOTE: The named-parameter INSERT relies on model_dump() key names matching DDL column names 1:1 (D-012 guarantee). No string interpolation anywhere.

    Public function get_events_for_run(conn, run_id: str) -> list[AuditEvent]:
      Execute: SELECT * FROM audit_events WHERE run_id=? ORDER BY timestamp ASC with (run_id,).
      Build AuditEvent from each row using dict(zip(cols, row)) pattern — re-validates on read.
      Return list of AuditEvent objects.

    Public function export_jsonl(conn, run_id: str, dest: io.TextIOBase | None = None) -> str:
      Execute same SELECT as get_events_for_run.
      For each row: construct AuditEvent, call event.model_dump_json(), append to lines list, write to dest with "\\n" suffix if dest is not None.
      Return "\\n".join(lines).

    No class, no global state, no connection management — conn and lock are always injected by callers (dependency injection pattern; connection management is the plugin's responsibility in Wave 2).

| hook callback → emit() | Hermes hook passes tool_name, args, result (arbitrary strings); these cross into ATLAS persistence boundary |
