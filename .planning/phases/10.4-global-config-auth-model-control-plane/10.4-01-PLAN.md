<threat_model>
| Threat | Mitigation |
|--------|------------|
| Lost concurrent update | expected revision checked inside OS lock |
| Crash-stale lock | OS-handle advisory lock auto-releases on process death |
| Brief world-readable temp | O_EXCL temp created 0600 before bytes are written |
| Windows replace race | lock is the serialization guarantee; replace is not overclaimed |
| Secret smuggling into config | frozen validator permits only empty/`env:` reference |
</threat_model>
