### `services/agent-runtime/atlas_runtime/surface_session_service.py` (service, lifecycle state machine)

**Analog:** `services/agent-runtime/atlas_runtime/run_service.py` (mirror exactly).

**Pydantic-first write + atomic SELECT-precondition transition + emit-after-lock** — the canonical session-write shape, copied from `run_service.py:46-93` (`start_run`) and `:114-142` (`complete_run`):
```python
# Source: run_service.py lines 46-91 — Pydantic-first create
run = Run(mission_id=mission_id, session_id=session_id, agent_runtime=agent_runtime)  # construct+validate BEFORE SQL
run_row = run.model_dump()
now = datetime.datetime.now(datetime.timezone.utc).isoformat()
with lock:
    with conn:                                   # TOCTOU-safe: SELECT + INSERT in one block
        row = conn.execute("SELECT status FROM missions WHERE id=?", (mission_id,)).fetchone()
        if row is None:
            raise ValueError(f"Mission {mission_id!r} not found")
        if row[0] != "pending":
            raise ValueError(f"Cannot start run for mission in state {row[0]!r}")
        conn.execute("INSERT INTO runs(...) VALUES (...)", run_row)
# Lock released — now safe to call emit() (which acquires lock internally)
emit(conn, lock, run_id=run.id, event_type="tool_call", session_id=session_id,
     data={"transition": "started", "mission_id": mission_id})
```
```python
# Source: run_service.py lines 115-142 — guarded transition
with lock:
    with conn:
        row = conn.execute("SELECT status FROM runs WHERE id=?", (run_id,)).fetchone()
        if row is None:
            raise ValueError(f"Run {run_id!r} not found")
        if row[0] != "running":                  # ← the transition guard to replicate
            raise ValueError(f"Cannot complete run in state {row[0]!r}")
        conn.execute("UPDATE runs SET status=?, finished_at=?, summary=? WHERE id=?", (...))
# lock released
emit(conn, lock, run_id=run_id, event_type="tool_call", data={"transition": status, "summary": summary})
```

**State machine guard (Pattern 6):** replace the single `if row[0] != "running"` with a table check `if row[0] not in _ALLOWED_FROM[target_state]: raise ValueError(...)`. Define `_ALLOWED_FROM` mapping for `starting→active→suspended→resuming→cancelling→completed|failed` plus `reclaimed`; terminal states have no outgoing transitions. Every transition emits via `emit()`.

**CRITICAL — Pitfall 6 (deadlock):** `emit()` re-acquires `lock` internally. Do the SQL transition inside `with lock:`, **release**, then call `emit()` AFTER. Never `emit()` inside the lock block. This is exactly why `run_service` emits after the `with lock:` block.

**Audit event_type:** pass new literals (e.g. `surface_session_started`, `surface_session_reclaimed`, `run_cancelled`, `permission_transition`) — they must first be added to the `core.py` Literal (see edit below). `emit()`'s `event_type` param is typed `str` and forwards anything the model accepts.

### `services/agent-runtime/atlas_runtime/workspace_service.py` (service, fail-closed path resolver)

**Analog:** `services/agent-runtime/atlas_runtime/project_service.py` (`_resolve_dir`, `ProjectError`, `get_project`).

**Typed error class + resolve-then-contain** — extend the `ProjectError(ValueError)` + `.expanduser().resolve()` convention from `project_service.py:25-33`:
```python
# Source: project_service.py lines 25-33
class ProjectError(ValueError):
    """Raised for invalid project inputs (bad path, missing folder, collision)."""

def _resolve_dir(root_path: str) -> pathlib.Path:
    if not root_path or not root_path.strip():
        raise ProjectError("root_path must not be empty")
    return pathlib.Path(root_path).expanduser().resolve()  # ← .resolve() collapses symlinks (reuse)
```
**Project-root lookup** — reuse `project_service.get_project` (`:102-112`); a missing project → typed `unregistered`:
```python
# Source: project_service.py lines 102-112
def get_project(conn, project_id) -> Optional[Project]:
    cursor = conn.execute("SELECT * FROM projects WHERE id=?", (project_id,))
    row = cursor.fetchone()
    return None if row is None else Project(**dict(zip([d[0] for d in cursor.description], row)))
```
**Containment guard (Pattern 2, SURF-02/03)** — the new logic to add (no exact analog; build per RESEARCH):
```python
class WorkspaceError(ValueError): ...   # reason ∈ {"traversal","symlink_escape","stale_root","unregistered"}

def assert_contained(root: str, target: str) -> str:
    root_p = pathlib.Path(root).expanduser().resolve()
    if not root_p.exists():
        raise WorkspaceError("stale_root")           # SURF-03: NO auto-reregister
    target_p = pathlib.Path(target).expanduser().resolve()
    try:
        common = pathlib.Path(os.path.commonpath([str(root_p), str(target_p)]))
    except ValueError:                               # Pitfall 1: mixed Windows drives → fail closed
        raise WorkspaceError("traversal")
    if common != root_p:
        raise WorkspaceError("symlink_escape" if _had_symlink(target) else "traversal")
    return str(target_p)
```
**Global root (Pitfall 4 — must agree with DB home):** resolve global workspace root from the SAME source as `db.DEFAULT_DB_PATH` = `pathlib.Path.home() / ".atlas"` (ATLAS_HOME-aware). Confirmed in `db.py:25`. Add one shared resolver so workspace root and DB home never diverge. Cross-project write denial must be audited via `emit()` (AUD-01).

### `services/agent-runtime/atlas_runtime/agents/native.py` (EDIT — poll cancel token)

**Analog:** self. Checkpoint is the watchdog join in `execute` (`:248-261`).

**Cooperative checkpoint (Pattern 7, Pitfall 3 — NOT a hard kill):** replace the single long `worker.join(self._max_runtime_s)` with a poll-join loop that also checks the cancel token between intervals:
```python
# Source: native.py lines 242-261 — the watchdog to make cancel-aware
def _drive() -> None:
    try:
        result_holder["result"] = agent.run_conversation(prompt)
    except BaseException as exc:
        error_holder["error"] = exc
worker = threading.Thread(target=_drive, name=f"native-run-{run_id[:8]}", daemon=True)
worker.start()
worker.join(self._max_runtime_s)        # ← change to poll-join: loop join(interval) checking cancel_token.is_set()
if worker.is_alive():
    self._safe_emit(conn, lock, run_id, event_type="failure",
                    data={"runtime": "native", "stop_reason": "max_runtime_exceeded"})
    return RunOutcome(status="failed", summary=..., stop_reason="max_runtime_exceeded", ...)
```
On `cancel_token.set()`: stop at the next checkpoint, `_safe_emit(... event_type="run_cancelled" ...)`, and return `RunOutcome(status="failed", summary="cancelled", stop_reason="cancelled")`. A2 caveat: the single `run_conversation` call is opaque/blocking — cancellation takes effect at the next checkpoint (between turns / tool gate / subprocess wait), never mid-call. Subprocesses spawned by tools ARE killable via `proc.terminate()` in a poll-loop. D-001: do NOT edit the foundation; any per-turn hook must already exist.

### Atomic transition (TOCTOU-safe + emit-after-lock)
**Source:** `services/agent-runtime/atlas_runtime/run_service.py` lines 115-142
**Apply to:** `surface_session_service.py` (every transition), `reconcile_orphans()`, workspace denial paths.

### Audit emission (single redacted write boundary, AUD-01)
**Source:** `services/agent-runtime/atlas_runtime/audit_service.py` lines 63-128
**Apply to:** every session lifecycle change, workspace denial, cancellation, permission transition.
`emit(conn, lock, *, run_id, event_type, session_id=..., data=...)` — `data` is JSON-serialized + redacted via `SECRET_PATTERNS` inside `emit()` (D-013 + AUD-01). The `AuditEvent` is constructed (and `event_type` Literal-validated) BEFORE any INSERT. Do not hand-roll redaction; do not write `audit_events` directly.

### Frozen-contract + id-gen + datetime serializer (D-013)
**Source:** `packages/atlas-core/atlas_core/schemas/core.py` lines 221-237; `agent_contract.py` lines 50-51
**Apply to:** `SurfaceSession`, `SurfaceEvent`, `SessionState`/`EventKind` Literals.
`model_config = ConfigDict(frozen=True, str_strip_whitespace=True[, extra="forbid"])`; `id` = `Field(default_factory=lambda: str(uuid.uuid4()))`; every `datetime` field gets a `@field_serializer(...)` returning `.isoformat()`. No `dict[str, Any]`, no `pathlib.Path` in any public field; payloads are JSON strings.

### Path resolution + typed errors
**Source:** `services/agent-runtime/atlas_runtime/project_service.py` lines 25-33, 102-112
**Apply to:** `workspace_service.py`.
`.expanduser().resolve()` (collapses symlinks — already the convention) before any containment check; a `ValueError` subclass per failure reason; project lookup via `get_project`.

### Immutable terminal records via SQL trigger
**Source:** `infra/migrations/0015_agent_contract_snapshots.sql` lines 12-22
**Apply to:** `0016_surface_sessions.sql` (optional terminal-state guard). Keep `BEGIN…END` intact — `db.py` runs `executescript`; `;`-splitting trigger bodies breaks them.

### Resume after disconnect/restart
**Source:** `services/agent-runtime/atlas_runtime/agent_contract_service.py` lines 187-213 (`load_contract`, `replay_contract`) + `RunContractSnapshot`/`ResumeSnapshot`
**Apply to:** SURF-05 resume. Rebuild session identity/context by replaying the existing immutable snapshot (network-free, deterministic). Do not invent a new snapshot format.

### Liveness (RECOMMENDED stdlib default)
**Source:** RESEARCH Pattern 5 (no in-repo analog — the codebase has no liveness precedent).
**Apply to:** `reconcile_orphans()` + heartbeat. Use `owner_token` (uuid) + `heartbeat_at` TTL (e.g. 3× heartbeat interval); a session is owned-alive iff `heartbeat_at` within TTL. PID-reuse-safe, Windows-safe, testable without process introspection. Do NOT use `os.kill(pid, 0)` (broken on Windows — `EINVAL` not `ESRCH`). `psutil` (already installed, 7.2.2) is optional hardening only if sub-TTL latency is required — flag as a planner decision (Open Question 1).
