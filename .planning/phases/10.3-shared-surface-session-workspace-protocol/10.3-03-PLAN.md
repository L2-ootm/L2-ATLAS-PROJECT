<objective>
Deliver the single normalized event contract that all surfaces (WebUI via gateway SSE, TUI via an
in-process iterator) will consume. One pure function projects immutable `audit_events` rows plus the
terminal `RunOutcome` into a `SurfaceEvent` stream with a discriminated `kind` and a monotonic
PER-SESSION `seq`. Because a SurfaceSession owns 0..N runs (SURF-01), the seq space must span all of a
session's runs — so this plan adds a per-session ordered read source `get_events_for_session` and feeds
the normalizer from it (the existing per-RUN `get_events_for_run` cannot support cross-run reconnect
gap-detection). Phase 10.3 scope is the contract + normalizer + conformance tests only — TUI/WebUI
rendering is wired in 10.6/10.7.

Purpose: SURF-04 (one normalized stream covering text, reasoning, tool calls/results, tasks/subagents,
retries, retrieval, approvals, errors, completion; per-session monotonic seq) + AGNT-01 (read-projection,
no new bus) + AUD-01 (events derive from the audited ledger).
Output: `surface_events.py`, the `get_events_for_session` addition to `audit_service.py`, and
`test_surface_events.py` (kind-coverage matrix + per-session contiguous-seq across two runs +
completion-carries-RunOutcome).

Depends on plan 01 for the `SurfaceEvent`/`EventKind` types and the new `core.py` audit literals.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| audit ledger → surface event stream | Persisted audit rows cross into a client-facing normalized projection |
| normalizer → transports (SSE / iterator) | Same projection consumed by multiple surfaces |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10.3-EVT-BUS | Tampering / Elevation | new event bus temptation | mitigate | normalizer is a pure read-projection over immutable `audit_events`; no pub/sub, no DB writes (AGNT-01) — enforced by a no-writes grep gate |
| T-10.3-EVT-AUTH | Information Disclosure / Elevation | cross-surface event delivery | mitigate | SurfaceEvent carries no decision authority; permission routing deferred to 10.5; gateway only forwards normalizer output (D-022) |
| T-10.3-EVT-LEAK | Information Disclosure | payload_json from audit data | mitigate | audit `data` is already redacted by `emit`/SECRET_PATTERNS at write time; normalizer reuses the redacted string and never re-derives raw payloads |
| T-10.3-EVT-REPLAY | Repudiation / replay confusion | reconnect gap-detection | mitigate | monotonic CONTIGUOUS per-session seq (over `get_events_for_session`) + `replay_since(last_seq)` gives deterministic gap detection across all the session's runs over immutable, time-ordered rows |
| T-10.3-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages — stdlib only |
</threat_model>
