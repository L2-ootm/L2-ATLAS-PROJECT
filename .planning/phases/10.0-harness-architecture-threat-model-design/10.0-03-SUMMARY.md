## What Was Built

### OAUTH_CALLBACK_THREAT_MODEL.md

A deferred-OAuth gate spec (Phase 7 shape). OAuth is explicitly deferred for v1.1 (SEC-05 locked); this document is the inheritance artifact for a future spike. Contains:

- 9 numbered hard-gate requirements (PKCE-S256, constant-time state, one-shot loopback listener at 127.0.0.1:0, exact redirect match, no-secret-in-logs, short timeout, loopback-http-acceptable, manual-paste fallback)
- HARD GATE: `hmac.compare_digest` for state comparison before processing `code` — the classic loopback-hijack vector is a naive `==`
- stdlib implementation hint (`http.server` bound to `("127.0.0.1", 0)`, single-request handler, `server.shutdown()`)
- STRIDE threat register for the OAuth surface
- Port-flexibility precondition (RFC 8252 §7.3 — not all providers support variable loopback port)
- Hand-off contract for the future spike (acceptance criteria, required tests)

### NATIVE_IPC_THREAT_MODEL.md

A Tauri 2 IPC threat model for the Phase 10.5 native shell. Contains:

- Tauri 2 deny-by-default capability model (per-command `allow-<cmd>` permissions, capability files scoped to windows)
- LANDMINE 5 HARD GATE: PTY transports keystrokes/bytes to a pre-spawned `atlas tui` child — never a command/argv string from the webview (NAT-03)
- Full IPC command inventory table (command / params / caller / privilege / threat / mitigation):
  - `get_readiness()` — read-only redacted status, no secrets
  - `pty_open(profile_id)` — validated enum, spawns fixed `atlas tui --profile <id>`
  - `pty_write(bytes)` — raw keystroke bytes to PTY stdin, child is `atlas tui` never a shell
  - `pty_resize(rows, cols)` — clamped to sane bounds
  - `open_external(url)` — Tauri 2 scope scheme allowlist (https/mailto only)
- Gateway posture: `atlas-gateway` binds `127.0.0.1:8484`, CORS allows `tauri://localhost`, is NOT an IPC surface
- PTY plugin selection deferred to Phase 10.5 with package-legitimacy audit requirement
- Reference capability file shape for `src-tauri/capabilities/main.json`
- STRIDE threat register for the IPC surface (8 rows)
- Hand-off contract for Phase 10.5
