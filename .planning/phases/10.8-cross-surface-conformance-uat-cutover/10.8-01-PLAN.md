<objective>
Build the shared conformance suite (success criterion 1): the same reference missions run
against all three surfaces' client seams and the normalized results must be equivalent.
Extend the 10.7 conformance assets — surface_event_parity.json, the cross-surface permission
conformance tests, the gateway contract tests — instead of inventing a parallel harness.
</objective>

<tasks>

1. **Reference mission fixture set.** Derive 8–12 missions from the Phase 10.2 promotion-gate
   scenarios and golden workflows: plain completion, tool call with audited result, Brain/wiki
   retrieval with provenance, permission ask→approve, permission ask→deny, cancellation
   mid-run, disconnect/resume, and config-change visibility. Each fixture pins prompt,
   expected normalized event kinds (from surface_event_parity.json), expected terminal status,
   and expected audit event types. Version the file (`"version": 1`).

2. **Per-surface execution seams.** The runner drives each surface at its client boundary,
   not by scraping rendered UI:
   - Go TUI: `internal/client` against the live gateway (extend client_test.go patterns).
   - atlas-terminal: the adapter fetch handle (`createAtlasFetchHandle`) — same seam the
     existing chat-loop tests use.
   - Cockpit: gateway REST/SSE via the same routes `src/lib/api.ts` calls.
   All three write one evidence JSON per mission: ordered normalized events, tool calls,
   provenance source ids, permission decisions with nonces, audit rows, final status.

3. **Equivalence assertions.** `test_cross_surface_conformance.py` loads the evidence files
   and asserts cross-surface equality after normalization (ids and timestamps masked, order
   preserved for order-guaranteed kinds). Diffs must name the first divergent field.

4. **Wire into CI-shape.** One entry point (`python scripts/conformance/run_cross_surface.py
   --all`) that boots the gateway via `gateway_control` (fresh, PID-tracked, never a leftover
   process), runs the battery, writes evidence under `.planning/phases/10.8-*/evidence/`, and
   exits nonzero on any divergence.

</tasks>
