## Deviations from Plan

### Auto-fixed Issues

**1. Split pure CLI logic from the JSX entrypoint**
- **Found during:** Task 2 test execution.
- **Issue:** The pinned Solid package exposes a type-only JSX runtime to Bun test imports.
- **Fix:** Moved option parsing/snapshot behavior into renderer-free `cli.ts`.
- **Verification:** Eight Bun tests pass without loading the renderer.
- **Committed in:** `51eca63`

**2. Added deterministic post-build forbidden-term scrub**
- **Found during:** Compiled artifact scan.
- **Issue:** Bun embeds an unrelated package-name table containing prohibited donor strings.
- **Fix:** Equal-length UTF-8/UTF-16 replacements driven by the external term file.
- **Verification:** The modified executable runs, matches source snapshot, and passes artifact scan.
- **Committed in:** `51eca63`

**3. Corrected dependency-directory exclusion**
- **Found during:** Phase-wide scan after installation.
- **Issue:** Root-relative globbing did not exclude nested `node_modules`.
- **Fix:** Changed exclusions to recursive globs and added a regression fixture.
- **Verification:** Installed dependencies are ignored while package metadata and artifacts remain scanned.
- **Committed in:** `98c84ec`

**4. Recorded a conditional memory ceiling**
- **Found during:** Task 3 baseline.
- **Issue:** OpenTUI/Bun measured 230.24 MiB idle working set versus the initial 150 MiB target; a core-only diagnostic showed a comparable substrate floor.
- **Fix:** Retained the miss in evidence, set a conditional <=240 MiB intake ceiling, and made Phase 10.6 remeasurement/decision mandatory.
- **Verification:** All other blocking budgets pass.
- **Committed in:** `98c84ec`
