### 2. Off-roadmap subsystem moved off main (review blockers B2/B3)

A module catalog + installer + L2-BOT integration runtime had been built outside the
roadmap and left fully uncommitted in the working tree. It contradicts Phase 8 judge
item 12 (don't build the module system speculatively), jumps to Phase 10 (installer) and
v2.0 (L2-BOT channels), and raises an unresolved D-022 new-Python-service question.

Moved verbatim to branch **`feat/integrations-module-catalog`** (commit `9ea990b`), then
removed from `main`'s working tree. Nothing was deleted — the branch holds the full
content (24 files, incl. `services/integration-runtime/` with its 16 passing tests) plus
the `atlas` CLI `integrations` wiring and the OVERVIEW/MODULE_SYSTEM docs. `main`'s
`cli/main.py` and `docs/architecture/OVERVIEW.md` are back at their committed state.

**Do not merge that branch into main until an explicit phase owns the work and the D-022
Python-exception question is decided.**
